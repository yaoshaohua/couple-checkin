# 情侣打卡小程序

一个有趣的情侣互动应用，记录爱意和怨气积分，用积分兑换任务惩罚对方。

## 功能特性

### 1. 积分系统
- **爱意积分**：记录开心、甜蜜的时刻
- **怨气积分**：记录生气、吵架的时刻
- 支持快捷打卡和自定义打卡

### 2. 打卡记录
- 查看所有打卡记录
- 按类型筛选（全部/爱意/怨气）
- 删除记录并退还积分
- 统计数据展示

### 3. 任务兑换
- 使用怨气积分兑换任务
- 内置多种任务选项（洗碗、做饭、按摩等）
- 支持自定义任务
- 任务完成标记
- 已完成任务历史

### 4. 个人设置
- 编辑情侣昵称
- 数据统计总览
- 导出数据（待实现）
- 清空数据

## 技术栈

- **框架**：uni-app
- **状态管理**：Vuex
- **数据存储**：uni.storage（本地存储）
- **UI设计**：渐变色卡片设计，粉色系主题

## 快速开始

### 1. 安装依赖

```bash
cd WebCloud.MiniApp.Couple
npm install
```

### 2. 配置小程序 AppID

在 `manifest.json` 中配置你的微信小程序 AppID：

```json
{
  "mp-weixin": {
    "appid": "你的AppID"
  }
}
```

### 3. 运行项目

```bash
# 开发环境
npm run dev:mp-weixin

# 生产构建
npm run build:mp-weixin
```

### 4. 导入微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `dist/dev/mp-weixin` 目录
3. 预览和调试

## 使用说明

### 打卡流程

1. **快捷打卡**：在首页选择预设的情绪选项快速打卡
2. **自定义打卡**：点击"自定义打卡"按钮，输入详细原因和积分
3. **查看记录**：在"记录"页面查看所有打卡历史

### 任务兑换流程

1. 积累足够的怨气积分
2. 在"任务"页面选择要兑换的任务
3. 确认兑换后，任务进入"待完成"列表
4. 对方完成任务后，点击"标记完成"

### 自定义任务

1. 点击"自定义任务"按钮
2. 选择任务图标
3. 输入任务名称和描述
4. 设置所需积分
5. 保存后即可使用

## 数据说明

所有数据存储在本地 `uni.storage` 中，包括：

- 爱意积分
- 怨气积分
- 打卡记录
- 任务列表
- 兑换记录
- 用户信息

## 自定义配置

### 修改主题色

在 `static/css/common.css` 中修改全局颜色变量：

```css
/* 主色调 */
background: linear-gradient(135deg, #FF6B9D 0%, #FFA07A 100%);
```

### 添加预设任务

在 `store/index.js` 的初始化数据中添加：

```javascript
tasks: [
  { id: 1, name: '任务名称', cost: 3, icon: '🍽️', description: '任务描述' },
  // ... 添加更多任务
]
```

## 注意事项

1. 本项目使用本地存储，数据仅保存在当前设备
2. 清空数据后无法恢复，请谨慎操作
3. 建议定期导出数据备份（功能待开发）
4. tabBar 图标需要自行准备，放在 `static/tabbar/` 目录下
